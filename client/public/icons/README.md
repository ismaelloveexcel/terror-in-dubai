# PWA Icons

Place your app icons here:

- `icon-192x192.png` - 192x192 pixels
- `icon-512x512.png` - 512x512 pixels
- `apple-touch-icon.png` - 180x180 pixels (for iOS)

## Design Recommendations

Use the game's theme colors:
- Background: #0a0a12 (dark navy)
- Primary: #00ffcc (teal glow)
- Accent: #cc0000 (danger red)

The icon should feature the game logo or a stylized "SI" for "Save Ismael" with a glowing effect.
